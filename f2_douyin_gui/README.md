# F2抖音数据采集工具 - GUI界面

这是一个基于Tkinter开发的图形用户界面，用于快捷调用f2抖音数据采集工具。

## 功能特点

### 📋 完整命令支持
- ✅ URL链接输入 (-u)
- ✅ 下载模式选择 (-M)：one, post, like, collection, collects, mix, live
- ✅ 显示语言选择 (-l)：zh_CN, en_US
- ✅ 保存选项：视频原声(-m)、视频封面(-v)、视频文案(-d)
- ✅ 单独文件夹保存 (-f)
- ✅ 自定义保存路径 (-p)
- ✅ Cookie输入 (-k)
- ✅ 日期区间筛选 (-i)
- ✅ 网络设置：超时时间(-e)、重试次数(-r)、并发连接(-x)
- ✅ 任务设置：最大任务数(-t)、最大下载数(-o)、每页作品数(-s)

### 📁 文件管理
- 📂 工作目录选择器
- 📂 保存路径选择器
- 📝 实时输出显示
- 📋 输出内容复制功能
- 🗑️ 输出内容清空功能

### 🎨 界面设计
- 现代化的Tkinter界面
- 响应式布局设计
- 清晰的分组显示
- 实时状态反馈

## 安装要求

### 必需软件
1. **Python 3.6+** - 推荐Python 3.8或更高版本
2. **f2抖音数据采集工具** - `pip install f2`
3. **Tkinter** - Python标准库，通常已预装

### 安装步骤

1. **安装f2工具**
```bash
pip install f2
```

2. **安装Tkinter**（如果未预装）
```bash
# Ubuntu/Debian
sudo apt-get install python3-tk

# macOS
brew install python-tk

# Windows
# 通常随Python安装，如未安装请重新安装Python并勾选Tkinter
```

## 使用方法

### 快速启动
```bash
# 直接运行主程序
python f2_douyin_gui.py
```

### 操作步骤

1. **选择工作目录**
   - 点击"选择目录"按钮选择工作目录
   - 程序将在此目录下执行f2命令

2. **填写URL链接**
   - 在"URL链接"输入框中填入抖音链接
   - 根据选择的下载模式，链接类型不同：
     - one: 单个作品链接
     - post: 主页链接
     - like: 主页链接（点赞作品）
     - collection: 主页链接（收藏作品）
     - collects: 收藏夹链接
     - mix: 合集链接
     - live: 直播链接

3. **选择下载模式**
   - 在下拉菜单中选择合适的下载模式

4. **配置保存选项**
   - 勾选需要保存的内容（原声、封面、文案）
   - 选择是否使用单独文件夹保存

5. **设置高级选项**（可选）
   - 自定义保存路径
   - 输入Cookie（用于需要登录的操作）
   - 设置日期区间筛选
   - 调整网络和任务参数

6. **执行采集**
   - 点击"执行采集"按钮开始下载
   - 在输出区域查看实时进度
   - 可使用"清空输出"和"复制输出"管理输出内容

## 配置说明

### 网络设置
- **超时时间(-e)**：网络请求超时时间，默认30秒
- **重试次数(-r)**：请求失败后的重试次数，默认3次
- **并发连接(-x)**：同时建立的网络连接数，默认5个

### 任务设置
- **最大任务数(-t)**：异步任务池大小，默认10个
- **最大下载数(-o)**：最多下载的作品数量，0表示无限制
- **每页作品数(-s)**：从接口获取的每页作品数，建议不超过20

### 日期区间格式
- 格式：`YYYY-MM-DD|YYYY-MM-DD`
- 示例：`2024-01-01|2024-12-31`
- 下载所有作品：输入`all`

## 注意事项

### 🔒 安全提醒
- Cookie包含您的登录信息，请妥善保管
- 不要在公共设备上保存Cookie
- 建议定期更新Cookie

### ⚡ 性能优化
- 根据网络状况调整并发连接数
- 大文件下载时适当增加超时时间
- 大量作品下载时可分批次进行

### ❗ 常见问题

**Q: 为什么无法下载某些作品？**
A: 可能原因：
- 作品已被删除或隐私设置
- Cookie过期或无效
- 网络连接问题
- 抖音API限制

**Q: 下载速度很慢怎么办？**
A: 尝试：
- 减少并发连接数
- 增加超时时间
- 检查网络连接
- 避开网络高峰期

**Q: 程序闪退或无响应？**
A: 尝试：
- 检查Python版本是否兼容
- 重新安装依赖包
- 以管理员身份运行
- 检查系统资源占用
