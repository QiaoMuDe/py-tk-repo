# F2抖音数据采集工具 - CustomTkinter版本

这是使用CustomTkinter库重构的F2抖音数据采集工具GUI界面，提供了更现代化的外观和用户体验。

## 功能特点

- 现代化的界面设计，支持亮色/暗色主题切换
- 使用选项卡组织界面，更加清晰直观
- 保留了原版本的所有功能
- 响应式布局，适应不同屏幕尺寸

## 安装依赖

```bash
pip install -r requirements_ctk.txt
```

## 运行程序

```bash
python f2_douyin_gui_ctk.py
```

## 界面说明

### 基本设置选项卡
- 工作目录选择
- URL链接输入
- 下载模式和显示语言设置
- 保存选项（视频原声、封面、文案、文件夹）
- 保存路径选择
- Cookie输入

### 高级设置选项卡
- 日期区间设置
- 网络设置（超时时间、重试次数、并发连接）
- 任务设置（最大任务数、最大下载数、每页作品数）

### 输出设置选项卡
- 输出方式选择（GUI显示/原生窗口显示）
- 执行输出区域
- 清空输出和复制输出功能

## 与原版本的区别

1. **界面风格**：使用CustomTkinter库，提供更现代化的界面
2. **布局组织**：使用选项卡组织不同功能区域，更加清晰
3. **控件样式**：所有控件使用CustomTkinter的现代化控件
4. **颜色主题**：支持亮色/暗色主题切换
5. **响应式设计**：界面布局更加灵活，适应不同屏幕尺寸

## 注意事项

1. 确保已安装Python和F2命令行工具
2. 首次使用前需要设置工作目录和Cookie
3. 所有功能与原版本保持一致，只是界面风格不同

## 主题切换

可以在代码中修改以下行来切换主题：

```python
# 设置外观模式
ctk.set_appearance_mode("System")  # 可选: "System", "Dark", "Light"

# 设置颜色主题
ctk.set_default_color_theme("blue")  # 可选: "blue", "green", "dark-blue"
```
